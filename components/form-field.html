Form Field

CUE Canvas form fields are intentionally minimal — styled vanilla HTML form elements customized with design tokens. They come in two styles (Border and Solid), three states (Default, Focus, Read Only), and two sizes (Default and XX-Small). All fields share the same base token set and are designed to be composed into Form Groups for layout.

Text Input

The base text input. Use the Border style for forms on white or light backgrounds. Use Solid when the field sits on a gray/elevated surface.

Form Groups

Form Groups compose a label, optional/required indicator, form field, and helper text into a consistent layout unit. Three layout patterns: Stack (label above field), Inline (label left, field right), and CRUD (tight label column for detail forms).

Combobox Selection Tag

Tags appear inside a combobox or multiselect field to represent selected values. Each tag has a dismiss button. Used in the Combobox and Multiselect components (see Date Picker page for more context).

--corner-form-field:                 6px;     /* px — border radius, never scales */
--padding-form-field-default:        0.75rem; /* 12px — default size padding */
--padding-form-field-medium:         0.5rem;  /* 8px  — XX-Small size padding */
--padding-form-field-small:          0.375rem;/* 6px  — compact padding */

/* Border style defaults */
--bg-form-field-default:             #ffffff;
--border-form-field-default:         #a3a3a3;
--content-form-field-placeholder:    #8a8a8a;
--content-form-field-filled-read:    #262626;

/* Solid style */
--bg-form-field-solid:               #ededed;
--content-form-field-solid-placeholder: #575757;

/* Focus state */
--bg-form-field-focus:               #ffffff;
--border-form-field-focus:           #0594d7; /* lighter brand blue — not --border-interactive */

/* Note: focus border #0594d7 is distinct from --border-interactive (#2c70f4).
   Use #0594d7 for form field focus rings only. */

/* Base input */
.cue-input {
  display: flex; align-items: center; gap: 0.5rem;
  width: 100%; padding: var(--padding-form-field-default);
  border: 1px solid var(--border-form-field-default);
  border-radius: var(--corner-form-field);
  background-color: var(--bg-form-field-default);
  font-family: var(--font-family-body); font-size: 1rem;
  color: var(--content-form-field-placeholder); outline: none;
  transition: border-color 0.15s ease;
}

.cue-input:focus {
  border-color: var(--border-form-field-focus);
  color: var(--content-form-field-filled-read);
}

.cue-input:read-only {
  border-color: transparent;
  background-color: transparent;
  color: var(--content-form-field-filled-read);
}

.cue-input--solid {
  background-color: var(--bg-form-field-solid);
  border-color: transparent;
  color: var(--content-form-field-solid-placeholder);
}

.cue-input--xxsmall {
  padding: var(--padding-form-field-medium);
  font-size: 0.6875rem; /* 11px */
}

/* Checkbox / Radio / Toggle — shared selected color */
/* All use #03618d (primary brand) for checked state */
.cue-checkbox:checked,
.cue-radio:checked     { background-color: #03618d; border-color: #03618d; }
.cue-toggle:checked    { background-color: #03618d; }
