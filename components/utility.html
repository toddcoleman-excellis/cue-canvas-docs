Utility UI

Purpose-built components used in specific contexts throughout CUE Back Office. Not used frequently but essential when needed. Includes Pagination, Tree View, Display Selection Toggle, App Mode Toggle, and Rich Text Toolbar.

Pagination

Used to navigate paginated data sets. Supports numbered pages, Previous/Next label buttons, and arrow-only icon buttons. The active page uses the primary brand background. Items share a common border via negative margin overlap.

Tree View

A hierarchical list supporting up to 5 indent levels. Each level adds a 16px indent column. Expandable rows use a chevron_right icon that rotates 90° when open. Available with and without vertical connector lines. Click items to expand/collapse.

Display Selection Toggle

A compact toggle for switching between grid density views — 4-up (2×2) and 9-up (3×3). Used on list/grid views to let users adjust how many tiles are shown. The active icon is dark; inactive is grey. Click to switch.

Default size

9-up active

App Mode Toggle

An iOS-style toggle for switching between Light and Dark mode. Light mode: grey track, white thumb with sun icon. Dark mode: dark track, dark thumb with moon icon. Three sizes available. Click to switch modes.

Rich Text Toolbar

A full WYSIWYG formatting toolbar. Used in rich text input fields and content editors. Buttons are grouped into logical sets (Font, Style, Insert, Align, List, Indent) with negative-margin borders connecting adjacent items in each group. Two sizes: Default (30px) and Small Compact (26px).
