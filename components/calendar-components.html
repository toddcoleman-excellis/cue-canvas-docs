Calendar & Scheduler Components

These are the atomic building blocks used to assemble the Calendar and Resource Scheduler views in CUE Back Office. The Calendar supports daily and weekly meeting or time-block scheduling. The Resource Scheduler supports production line and personnel planning across time.

Day and Date Header

Column header block showing the day name and date number. The current day variant has a 4px amber bar across the top using the --current-day-active-indicator token.

Hour Block

The structural time slot of the calendar grid. Each block represents one hour, visually split into two 30-minute halves by a dashed border. Activity blocks are positioned absolutely over the hour grid. Available in Default and Vertical orientations, and Default or Current Day types.

User Avatar

24px circular avatar used in the bottom slot of calendar activity blocks. Uses the accent brand color as background with white initials. The Action type (primary brand blue) is used as the last item to add more attendees. Avatars overlap using a -4px right margin.

Resource / Personnel Block

Column header in the Resource Scheduler. Shows the resource or person's avatar, name, employment status tag, and an actions menu. Used in the fixed left column of the scheduler grid.

Scheduler Activity Block

An event block placed in the resource scheduler grid. Wider than the calendar activity block (248px vs 189px) and includes production details or serial numbers in the description slot. No avatar strip — resource assignment is handled at the row level.

Notes for tokens.css

The following calendar-specific tokens should be added to tokens.css when integrating into a project:

/* Components — Calendar */
--bg-hour-block:                      #ffffff;
--border-hour-block:                  #ededed;
--bg-hour-block-current-day:          #ffffff;
--border-hour-block-current-day:      #ededed;
--border-dashed-half-hour:            #ededed;
--border-dashed-half-hour-current-day: #ededed;
--current-day-active-indicator:       #fab632;

/* Components — Calendar Activity */
--bg-cal-activity-block:              #ffffff;
--border-cal-activity-block:          #ededed;
--bg-cal-activity-block-hover:        #ffffff;
--border-cal-activity-block-hover:    #8a8a8a;

/* Components — Resource Scheduler */
--bg-resource-sched-block:            #ffffff;
