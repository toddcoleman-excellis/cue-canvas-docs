Toast

A temporary notification panel that appears fixed at the bottom-right of the viewport. Used to confirm completed actions, surface system feedback, or alert users to events. Toasts auto-dismiss after a configurable duration or can be manually closed. Multiple toasts stack vertically.

Base

The base toast from Figma — white background, subtle border, Shadow/12. Title is 16px Bold, description is 14px Regular. The Close button is the neutral-stroke button style.

Title of action or feedback This is a description of the action or feedback that has occurred.

Status variants

Four status variants using the same color tokens as the Alert component. Each has a leading status icon. Add the variant modifier class to .cue-toast.

check_circle
Changes saved successfully Your order has been updated and confirmation sent to the customer.
info
Sync in progress Your catalog is syncing with the warehouse system. This may take a moment.
warning
Low inventory detected 3 SKUs are below minimum stock levels. Review your inventory report.
error
Failed to submit order A network error occurred. Please check your connection and try again.

Live demo

Click a button to trigger a toast in the bottom-right corner. Toasts auto-dismiss after 5 seconds and show a progress bar. Multiple toasts stack.

/**
 * showToast(type, title, desc, duration)
 *
 * type:     'neutral' | 'success' | 'info' | 'warning' | 'error'
 * title:    string
 * desc:     string (optional)
 * duration: ms before auto-dismiss (default: 5000, 0 = no auto-dismiss)
 */
function showToast(type, title, desc, duration) {
  duration = duration !== undefined ? duration : 5000;
  var stack = document.getElementById('toast-stack');
  if (!stack) return;

  var icons = {
    success: 'check_circle',
    info:    'info',
    warning: 'warning',
    error:   'error'
  };

  var toast = document.createElement('div');
  toast.className = 'cue-toast is-entering' + (type !== 'neutral' ? ' cue-toast--' + type : '');
  toast.setAttribute('role', 'alert');

  var iconHTML = icons[type]
    ? '<div class="cue-toast__icon"><span class="material-icons" aria-hidden="true">' + icons[type] + '</span></div>'
    : '';

  var descHTML = desc
    ? '<span class="cue-toast__desc">' + desc + '</span>'
    : '';

  var progressHTML = duration > 0
    ? '<div class="cue-toast__progress" style="animation-duration:' + duration + 'ms"></div>'
    : '';

  toast.innerHTML = iconHTML +
    '<div class="cue-toast__body">' +
      '<span class="cue-toast__title">' + title + '</span>' +
      descHTML +
    '</div>' +
    '<button class="cue-toast__close" type="button" onclick="dismissToast(this.closest(\'.cue-toast\'))">Close</button>' +
    progressHTML;

  stack.appendChild(toast);

  // Trigger enter animation
  requestAnimationFrame(function() {
    requestAnimationFrame(function() {
      toast.classList.remove('is-entering');
    });
  });

  // Auto-dismiss
  if (duration > 0) {
    setTimeout(function() { dismissToast(toast); }, duration);
  }
}

function dismissToast(toast) {
  if (!toast || toast.classList.contains('is-leaving')) return;
  toast.classList.add('is-leaving');
  setTimeout(function() { toast.remove(); }, 300);
}
