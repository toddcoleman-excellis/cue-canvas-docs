Table

Tables are the primary way data is structured and browsed in CUE Back Office. They support sorting, selection, inline actions, row expansion, filtering, and pagination. CUE uses three table types — Primary, Secondary, and Tertiary — each suited to a different context and density of use.

Table Types

Choose the table type based on context. Primary is the default for any main data view. Secondary is used for dense supporting data. Tertiary is used in tight spaces like modals.

Primary Table
The main data table used throughout CUE. Includes zebra-stripe rows, expanding row panels, full utility bar with search and actions, and top and bottom pagination.
Default
Secondary Table
Used to show more compact data — order lines, material info, related records. Zebra-striped but more vertically compact than the Primary Table.
Dense data
Tertiary Table
A simple, compact table with no zebra striping. Used in constrained spaces like modals or side panels. Can include a selected state. No utility bar.

Row Density

The table supports three density settings: Default (56px rows), Cozy (64px), and Compact (36px). Users can switch density via the settings icon in the utility bar. Density affects row height only — typography and padding remain consistent.

    row.classList.toggle('selected');
    // update checkbox state
    const cb = row.querySelector('.cue-table-checkbox');
    if (cb) cb.classList.toggle('checked', row.classList.contains('selected'));
  });
});

// Column sort (simple DOM sort)
function sortTable(colIndex) {
  const table = document.querySelector('.cue-table');
  const tbody = table.querySelector('tbody');
  const rows = Array.from(tbody.querySelectorAll('tr'));
  const dir = table.dataset.sortDir === 'asc' ? 'desc' : 'asc';
  table.dataset.sortDir = dir;
  table.dataset.sortCol = colIndex;

  rows.sort((a, b) => {
    const aText = a.cells[colIndex]?.textContent.trim() ?? '';
    const bText = b.cells[colIndex]?.textContent.trim() ?? '';
    return dir === 'asc'
      ? aText.localeCompare(bText, undefined, { numeric: true })
      : bText.localeCompare(aText, undefined, { numeric: true });
  });

  rows.forEach(row => tbody.appendChild(row));

  // Update sort icons
  document.querySelectorAll('.cue-sort-icon').forEach(icon => {
    icon.classList.remove('cue-sort-icon--asc', 'cue-sort-icon--desc');
  });
  const activeIcon = document.querySelector(`#sort-${colIndex}`);
  if (activeIcon) activeIcon.classList.add(`cue-sort-icon--${dir}`);
}
