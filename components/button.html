Icon Reference

Button

Buttons communicate actions users can take. They are typically placed throughout the UI in forms, dialogs, panels, and as standalone calls to action. Buttons are composed of three optional elements: a leading icon, a text label, and a trailing icon.

Custom Color

The Custom Color button type is flexible — use it when a semantic color (success, error, warning) needs to be expressed as a button. Override --btn-custom-bg and --btn-custom-content on the element.

  <span class="cue-btn__label">Export</span>

</button>

<button class="cue-btn cue-btn--link" type="button">
  <span class="cue-btn__label">View all</span>
  <span class="cue-btn__icon-trailing material-icons" aria-hidden="true">arrow_forward</span>

<!-- Neutral (No Padding) -->
<button class="cue-btn cue-btn--neutral-link" type="button">
  <span class="cue-btn__label">Dismiss</span>
</button>

<!-- Brand Accent -->
<button class="cue-btn cue-btn--brand-accent" type="button">
  <span class="cue-btn__label">Upgrade</span>
</button>

<!-- Custom Color (Danger example) -->
<button class="cue-btn cue-btn--custom" type="button"
  style="--btn-custom-bg: var(--danger-default); --btn-custom-content: #ffffff;">
  <span class="cue-btn__icon-leading material-icons" aria-hidden="true">delete</span>
  <span class="cue-btn__label">Delete</span>
</button>

<!-- Size modifier example -->
<button class="cue-btn cue-btn--primary cue-btn--small" type="button">
  <span class="cue-btn__label">Small Button</span>
</button>

<!-- Disabled -->
<button class="cue-btn cue-btn--primary" type="button" disabled>
  <span class="cue-btn__label">Unavailable</span>
</button>

<!-- Icon only — always include aria-label on the button -->
<button class="cue-btn cue-btn--neutral-stroke" type="button" aria-label="Delete item">
  <span class="cue-btn__icon-leading material-icons" aria-hidden="true">delete</span>
</button>

Accessibility

Attribute / PatternElementNotes
type="button" .cue-btn Always set explicitly to prevent accidental form submission.
aria-label .cue-btn (icon-only) Required when the button has no visible text label. Describes the action.
aria-hidden="true" .cue-btn__icon-* Icons are decorative. The button label or aria-label communicates the action.
disabled .cue-btn Native disabled attribute removes the button from tab order and announces state to screen readers.
:focus-visible .cue-btn Focus ring shown on keyboard navigation only. Uses --border-interactive (blue).
Contrast All types All button types meet WCAG AA 4.5:1 contrast ratio for text. Verify custom color overrides independently.
