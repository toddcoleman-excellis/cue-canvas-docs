Icon Reference

Alert

Alerts communicate important contextual information to the user — confirmations, notices, warnings, and errors. They are non-interactive by default and appear inline within page content. Each type uses a distinct color drawn from the primitive color scale to ensure immediate visual recognition.

Types

check_circle

Success

Your changes have been saved successfully.

info

Information

This is where the main content of the alert will go.

warning

Warning

This action may have unintended consequences. Please review before continuing.

error

Error

Something went wrong. Please try again or contact support.

Orientations

Stack (default) — title above body

info

Alert Title

This is where the main content of the alert will go.

Inline — title and body on one line

info

Alert Title

This is where the main content of the alert will go.

Design Tokens

Alert colors are drawn directly from the Color Primitives scale rather than the Global semantic tokens. This gives each alert type a lighter, tinted appearance suitable for inline content contexts.

TypeRolePrimitiveValue
Success Background green-50 #ebfaef
Border green-300 #86df9c
Text & Icon green-800 #155124
Info Background blue-50 #ecf2fe
Border blue-300 #81a9f8
Text & Icon blue-800 #06296f
Warning Background orange-50 #fff6e5
Border orange-300 #ffc561
Text & Icon orange-800 #613d00
Error Background red-50 #ffe0e0
Border red-300 #ff4d4d
Text & Icon red-800 #570000

Accessibility

Attribute / PatternElementNotes
role="alert" .cue-alert--error, --warning, --success Announces the alert to screen readers immediately when it appears in the DOM. Use for time-sensitive or important messages.
role="status" .cue-alert--info Politely announces the message without interrupting. Use for informational notices that are not urgent.
aria-hidden="true" .cue-alert__icon Icon is decorative — the alert type is communicated by the title text and role, not the icon alone.
Color contrast All types All text (800 stop) on background (50 tint) meets WCAG AA 4.5:1 contrast ratio.
Don't rely on color alone All types Each type uses a distinct icon and title text in addition to color to communicate meaning.
