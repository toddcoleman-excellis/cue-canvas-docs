Chat

The Communication and Chat feature is a concept-phase internal messaging module for CUE Back Office — similar in intent to Slack or Microsoft Teams. Components are documented here to establish the design language ahead of implementation.

build_circle Concept Phase

User Avatar

36px circle. Four types: single initial, two-letter initials, photo, and action (used for "add participant"). All use primary brand blue background.

Message Bubbles

Two bubble types. From (received from someone else) uses a light gray background with a bottom-left tail. To (sent by the current user) uses the primary brand blue with a bottom-right tail. The border radius asymmetry creates the speech bubble direction.

From — received

This is a message from a coworker or client to you. It comes with the ability to change the avatar and show the username, time, and date.

To — sent

This is a message from you to a client or coworker. It comes with the ability to change the avatar and show the username, time, and date.

Message Groups

A message group combines a bubble with the sender's avatar and the name / date / time metadata row. Received messages align left with avatar on the left. Sent messages align right with avatar on the right.

Message Input

The input area sits at the bottom of the chat thread. A formatting toolbar can be toggled via the "A" button — it slides open to show rich text controls. Attachment and mention buttons sit alongside the Send button in the actions row.

Closed — no toolbar

Open — with formatting toolbar

Message List Item

Conversation rows in the left panel. The current conversation has a white background with a 2px accent blue left border. Inactive conversations use a light gray background. The unread indicator is a 12px red dot — absent when read.
