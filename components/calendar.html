Layout notes

ElementValueNotes
Hour block height 120px Fixed. Each 30-min half = 60px. Used as the base unit for all time calculations.
Day header height 77px min-height Sticky, stays visible on scroll. Time label column header is empty (same height).
Activity block top dayHeaderH + (startHour × 120) + (startMin ÷ 60 × 120) Grid start hour is 7:00 AM in this example. Adjust gridStartHour to match your data.
Activity block height (durationMinutes ÷ 60) × 120 Minimum visible height should be ~32px (title + time only) even for very short events.
Activity block width ~90% of column width Leave a small right margin so the grid border is visible. Overlapping events should tile side-by-side.
Current time indicator top dayHeaderH + ((currentHour - gridStartHour) × 120) + (currentMin ÷ 60 × 120) opacity: 0.75. Lead segment width = time label column width. Circle starts at the current-day column boundary.
Container .cue-calendar uses .cue-card tokens White bg, #ededed border, 4px radius, 16px padding. Lives inside the page content area, not full-bleed.
