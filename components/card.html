Icon Reference

Card

Cards group related information into a contained surface. They appear throughout CUE Back Office for KPI summaries, entity overviews, customer details, and module navigation. All cards share the same base surface tokens but differ in anatomy and purpose.

Card — Large

The flagship card used for primary module navigation or feature summaries. Houses a 48px icon, title, description, optional action button, and an optional row of three KPI stat tiles below.

Card Icons

Cards use a set of 29 custom 48×48px SVG icons from the CUE Canvas Card Icons library. These are distinct from Material Icons and represent common B2B enterprise concepts — warehouse, factory, delivery, inventory, users, reports, and so on. In the documentation examples above, Material Icons are used as placeholders. In production, use the custom Card Icon SVGs from the Figma library.

Available icons include: Badge, Bookshelf, Briefcase, Briefcase-user, Building location, Cash, Delivery truck, Factory, Hand cart-1, Hand cart-2, Handshake, Id-card, Inventory box, Package, Package check 1, Package check 2, Package-2, Permissions, Receipt-long, Recall, Reports, Settings, Storefront, Tag, User settings, Users, Users-2, Users-circle, Warehouse.
