Breadcrumb

Breadcrumbs show the user's current location within the application hierarchy and provide a way to navigate back up through parent sections. The breadcrumb bar spans the full width of the content area and sits below the app top bar on every interior page of CUE Back Office.

Example

  1. Home keyboard_arrow_right
  2. Order Management keyboard_arrow_right
  3. Orders keyboard_arrow_right
  4. Order #10482

Note: in production the breadcrumb bar spans the full page width. Horizontal padding is inherited from the page layout wrapper.

Item types

Back — navigates to previous page

Home — first item in trail

Home keyboard_arrow_right

Link and Chevron — intermediate steps

Order Management keyboard_arrow_right

Current — end of trail, not a link

Order #10482

Code

<nav class="cue-breadcrumb" aria-label="Breadcrumb">
  <div class="cue-breadcrumb__content">

    <!-- Back button -->
    <button class="cue-breadcrumb__back" type="button" onclick="history.back()">
      <span class="cue-breadcrumb__back-icon material-icons" aria-hidden="true">arrow_back</span>
      Back
    </button>

    <!-- Pipe divider -->
    <span class="cue-breadcrumb__pipe" aria-hidden="true"></span>

    <!-- Trail -->
    <ol class="cue-breadcrumb__trail" aria-label="Page navigation">

      <!-- Home -->
      <li class="cue-breadcrumb__item">
        <a href="/dashboard" class="cue-breadcrumb__link">Home</a>
        <span class="cue-breadcrumb__chevron material-icons" aria-hidden="true">keyboard_arrow_right</span>
      </li>

      <!-- Intermediate links -->
      <li class="cue-breadcrumb__item">
        <a href="/orders" class="cue-breadcrumb__link">Order Management</a>
        <span class="cue-breadcrumb__chevron material-icons" aria-hidden="true">keyboard_arrow_right</span>
      </li>

      <!-- Current page — not a link -->
      <li class="cue-breadcrumb__item" aria-current="page">
        <span class="cue-breadcrumb__current">Order #10482</span>
      </li>

    </ol>
  </div>
</nav>

Accessibility

Attribute / PatternElementNotes
<nav> with aria-label .cue-breadcrumb Use a <nav> element with aria-label="Breadcrumb" to create a landmark for screen reader navigation.
<ol> for trail .cue-breadcrumb__trail An ordered list communicates the sequential hierarchy to screen readers.
aria-current="page" .cue-breadcrumb__item (last) Applied to the list item containing the current page label. Tells screen readers which item represents the current location.
Current is not a link .cue-breadcrumb__current Use a <span> not an <a> for the current page — it should not be navigable since the user is already there.
aria-hidden="true" .cue-breadcrumb__chevron, .cue-breadcrumb__pipe Decorative elements hidden from screen readers — the list structure communicates hierarchy.
:focus-visible .cue-breadcrumb__back, .cue-breadcrumb__link Visible focus ring on keyboard navigation using the interactive border color token.
