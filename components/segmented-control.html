Segmented Control

A set of mutually exclusive toggle buttons inside a shared pill-shaped container. Similar in function to tabs but rendered as an inline button group — closer to the iOS segmented control or ShadCN's Tabs component. Used for switching between views or filter states within a page section.

Simple — Compact

Container width fits its content. Buttons do not stretch. Active button gets a white background, subtle border, and a drop shadow that lifts it off the track. Click to switch segments.

Simple — Wide

Container spans full width. Each button stretches equally with flex: 1. Useful when the segmented control should fill a column or panel. Add .cue-seg--wide to the container.

With Icon — Compact

Adds a 14px leading icon to each segment. The icon inherits the button's text color, so it automatically shifts between moderate gray (inactive) and default dark (active). Use a filled Material Icon for the active state and outlined for inactive, or a single icon with color difference.

With Icon — Wide

Full-width variant with leading icons. Each segment stretches equally.

// Activate clicked segment, deactivate siblings
function activateSeg(btn) {
  const seg = btn.closest('.cue-seg');
  seg.querySelectorAll('.cue-seg__btn').forEach(function(b) {
    b.classList.remove('is-active');
    b.setAttribute('aria-selected', 'false');
  });
  btn.classList.add('is-active');
  btn.setAttribute('aria-selected', 'true');
}

// Optional: keyboard arrow navigation
document.querySelectorAll('.cue-seg').forEach(function(seg) {
  seg.addEventListener('keydown', function(e) {
    const btns = [...seg.querySelectorAll('.cue-seg__btn')];
    const idx  = btns.indexOf(document.activeElement);
    if (e.key === 'ArrowRight' && idx < btns.length - 1) {
      e.preventDefault(); btns[idx + 1].focus();
    } else if (e.key === 'ArrowLeft' && idx > 0) {
      e.preventDefault(); btns[idx - 1].focus();
    } else if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault(); activateSeg(document.activeElement);
    }
  });
});
