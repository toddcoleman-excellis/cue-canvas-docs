Icons

Icon Reference

Expand / Collapse

A surface module with a title header and toggleable content panel. Used throughout CUE Back Office to group related information — form sections, permissions, detail records, data points — under a named heading that can be expanded or collapsed independently. The content area is fully flexible: any HTML can go inside it.

Collapsed (default)

When collapsed only the header row is visible — title on the left, "Expand" button with unfold_more icon on the right. The module has a minimal height and no bottom padding gap.

Module Title

This is the expanded content area. Any component, form, data table, or layout can go here. The content area is fully flexible.

Expanded

When expanded the content area is visible below the header. The button switches to "Collapse" with the unfold_less icon. The 24px gap between the header and content body only renders when the panel is open.

Module Title

This is the expanded content area. Any component, form, data table, or layout can go here. The content area is fully flexible.

Stacked modules

Multiple Expand / Collapse modules stacked vertically with a 10px gap — the pattern used in the permissions section and detail pages. Each panel is independently expandable. Click any Expand button to try it.

function toggleExpand(toggleBtn) {
  const module = toggleBtn.closest('.cue-expand');
  const isOpen = module.classList.toggle('is-open');
  const label = toggleBtn.querySelector('.cue-expand__toggle-label');
  const icon  = toggleBtn.querySelector('.cue-expand__toggle-icon .material-icons');

  toggleBtn.setAttribute('aria-expanded', isOpen);
  label.textContent = isOpen ? 'Collapse' : 'Expand';
  icon.textContent  = isOpen ? 'unfold_less' : 'unfold_more';
}

// Keyboard: Enter or Space on the toggle button is handled natively
// since it's a <button> element.

// Optional: expand all / collapse all
function expandAll() {
  document.querySelectorAll('.cue-expand').forEach(module => {
    if (!module.classList.contains('is-open')) {
      const toggle = module.querySelector('.cue-expand__toggle');
      if (toggle) toggleExpand(toggle);
    }
  });
}

function collapseAll() {
  document.querySelectorAll('.cue-expand.is-open').forEach(module => {
    const toggle = module.querySelector('.cue-expand__toggle');
    if (toggle) toggleExpand(toggle);
  });
}
