Skeleton Loader

Skeleton loaders are placeholder UI shapes shown while content is loading. They reduce perceived wait time by giving users a structural preview of the content before it arrives.

Overview

Use skeleton loaders in place of spinners when the layout of the incoming content is known. They communicate structure and set expectations, making the interface feel faster and more predictable. Skeletons animate with a left-to-right shimmer to indicate activity.

Shapes

Skeletons support three shape variants via modifier classes. Match the skeleton shape to the content it represents.

Default (4px)
Rounded (8px)
Circle

Usage Guidelines

Guideline Detail
Match content shape Use skeleton shapes that closely match the dimensions of the actual content. A text line should be a thin rectangle; an avatar should be a circle of the same size.
Animate always The shimmer animation is required — a static skeleton reads as an error, not a loading state. Never disable the ::after animation.
Vary widths for text When representing multi-line text, use staggered widths (100%, 90%, 85%, 95%) to feel natural. Identical widths look mechanical.
Don't use for short waits If content resolves in under 300ms, skip the skeleton entirely — flashing a skeleton before immediate content creates more disruption than a blank moment.
Preserve layout Skeleton containers should match the height of their content targets so no layout shift occurs when content loads.
Accessibility Add role="status" and aria-label="Loading" to the skeleton wrapper so screen readers announce the loading state.

Code

<!-- Rectangle (default) -->
<span class="cue-skeleton" style="height: 14px; width: 200px;"
      role="status" aria-label="Loading"></span>

<!-- Rounded rectangle -->
<span class="cue-skeleton cue-skeleton--rounded" style="height: 120px;"
      role="status" aria-label="Loading"></span>

<!-- Circle (avatar) -->
<span class="cue-skeleton cue-skeleton--circle" style="width: 40px; height: 40px;"
      role="status" aria-label="Loading"></span>

<!-- Composed: profile list item -->
<div role="status" aria-label="Loading" style="display: flex; gap: 0.75rem; align-items: center;">
  <span class="cue-skeleton cue-skeleton--circle" style="width: 36px; height: 36px; flex-shrink: 0;"></span>
  <div style="display: flex; flex-direction: column; gap: 0.375rem; flex: 1;">
    <span class="cue-skeleton" style="height: 12px; width: 55%;"></span>
    <span class="cue-skeleton" style="height: 10px; width: 35%;"></span>
  </div>
</div>
.cue-skeleton {
  display: block;
  background-color: #ededed;
  border-radius: 4px;
  overflow: hidden;
  position: relative;
}

.cue-skeleton::after {
  content: "";
  position: absolute;
  inset: 0;
  background: linear-gradient(
    90deg,
    rgba(255,255,255,0) 0%,
    #ffffff 50%,
    rgba(255,255,255,0) 100%
  );
  animation: skeleton-shimmer 1.5s infinite;
  transform: translateX(-100%);
}

@keyframes skeleton-shimmer {
  0%   { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}

.cue-skeleton--circle  { border-radius: 50%; }
.cue-skeleton--rounded { border-radius: 8px; }
